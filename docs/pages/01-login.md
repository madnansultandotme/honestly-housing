# Login

Build email/password sign-in for Builder and Client roles. Include password reset link for Firebase Auth. After authentication, load user profile from users collection by uid and route based on role: Builder to 02a-builder-dashboard, Client to 02b-client-dashboard. Handle loading, auth error, and network error states. Design with clean card on white background (#FFFFFF), primary button in brass accent (#B8956A), secondary reset link in warm neutral gray (#8B8680). Apply rounded corners, premium spacing, soft taupe (#D4C4B0) accents.
